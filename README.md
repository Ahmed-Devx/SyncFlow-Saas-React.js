# 🚀 SyncFlow — Modern SaaS Landing Page

A fully responsive and modern **SaaS landing page** built with **React JS** and **Tailwind CSS**, designed using **Tailblocks** layouts and reusable React components.  
This project focuses on clean UI, scalable structure, and best practices used in real-world SaaS applications.

---

## 🌐 Live Demo
👉 **Netlify:**  https://jolly-sunburst-cf0e22.netlify.app/

 

---

## ✨ Features
- Modern SaaS landing page design
- Fully responsive (mobile, tablet & desktop)
- Reusable React components
- Props-based data rendering
- Clean and scalable folder structure
- Tailwind CSS utility-first styling
- Tailblocks-based professional sections
- Optimized UI for real-world SaaS products

---

## 🛠️ Tech Stack
- **React JS** (Vite)
- **Tailwind CSS**
- **Tailblocks**
- **React Icons**
- **JavaScript (ES6+)**
- **Netlify** (Deployment)

---

 

## 📌 Sections Included
- **Navbar** — Logo, navigation links, CTA button  
- **Hero Section** — Main headline, description & call to action  
- **Features Section** — Props-driven reusable feature cards  
- **How It Works** — Step-by-step workflow explanation  
- **Pricing Section** — Card-based pricing plans  
- **CTA Section** — Conversion-focused call to action  
- **Footer** — Social links & copyright info
